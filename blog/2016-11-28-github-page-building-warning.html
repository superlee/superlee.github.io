Github Page build warning

发表于 2016-11-28 |

昨天刚刚配置好github page的个人域名，今天就收到github邮件

The page build completed successfully, but returned the following warning:

Your site’s DNS settings are using a custom subdomain, www.chaozheng.li, that’s set up as an A record. We recommend you change this to a CNAME record pointing at superlee.github.io. For more information, see https://help.github.com/pages/.

For information on troubleshooting Jekyll see:

https://help.github.com/articles/troubleshooting-jekyll-builds

If you have any questions you can contact us by replying to this email.

简单来说就是要把域名的A记录变为CNAME

  • A记录是把域名和主机的IP地址绑定
  • CNAME 是把域名和hostname绑定

一般来说对于有明确固定IP的主机可以使用A记录，而对于IP地址可能会变的推荐使用CNAME。
